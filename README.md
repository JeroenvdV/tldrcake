# TLDRCake 🍰

TLDRCake is a summary format that lets you stop reading the moment you know enough.

It works by stacking summaries in growing sizes, where each one builds on the last. Nothing gets repeated. That first sentence already gives you the whole idea, and everything after it just adds more detail.

The problem it solves is simple: we now have AI that can write endlessly, but people still have limited time to read. Traditional summaries haven't caught up with that shift. A normal summary tries to stand on its own, which means it ends up repeating itself when you also read the full version. TLDRCake removes that waste. Every sentence you read is new information.

The layers grow in a specific pattern: 1, 2, 4, 7, 13, 25, and 40 sentences. Each one is about twice the size of the last. That ratio matches how people naturally decide to keep reading. If what you just read was worth it, you're willing to read a bit more, but not too much more. Picture a layered cake where each tier is thicker than the one above it, with a cherry on top. That cherry is your opening sentence. You decide how far down you want to go.

This works in almost any setting where different people need different amounts of detail. A manager reads three sentences and knows whether something matters. A specialist reads the whole thing and has enough to get started. Investors can scan dozens of proposals in minutes instead of hours. Companies can make internal communication clearer without asking anyone to change their tools or habits. All you need is the right instructions for an AI model and it turns any document into this format automatically. There is nothing to install, nothing to learn. The idea behind TLDRCake is that we have been focused on making writing faster and easier for years now. But reading is where people actually lose time. This format is a practical answer to that problem.

## Use TLDRCake

This repo includes a ready-to-use LLM skill file that you can plug into any AI workflow. The skill file contains a complete system prompt that instructs a language model to produce TLDRCake summaries from any input, whether that's a business proposal, a research paper, a product spec, or a rough idea someone typed out in five minutes.

To get started, grab the skill file from this repo and use it as a system prompt with your preferred language model. Paste any content into the user message and you'll get a TLDRCake back. No setup, no configuration, no learning curve.

If you find the format useful, start sharing TLDRCakes in your team communication, your documentation, your proposals. The more people encounter the format, the more natural it becomes. Every time you summarize something as a TLDRCake instead of a wall of text, you're respecting your reader's time. That's the whole point.
